# GlobalAmBC-DRL: Artefato SBRC 2026

## Descrição

Este repositório contém os artefatos experimentais do artigo:

"Um Novo Módulo Arquitetural de Controle Centralizado com DRL para Redes IoT Densas sem Bateria Baseadas em Retroespalhamento Ambiente"

O objetivo deste artefato é permitir a reprodução dos principais resultados apresentados no artigo, especialmente o comportamento temporal da taxa de sucesso da rede sob diferentes estratégias de controle.

---

## Estrutura do Repositório

- `data/` → dados utilizados nos experimentos
- `scripts/` → scripts de geração e execução
- `figures/` → figuras geradas
- `docs/` → documentação adicional

---

## Selos Considerados

Este artefato visa obter os seguintes selos:

- Artefatos Disponíveis (SeloD)
- Artefatos Funcionais (SeloF)
- Experimentos Reprodutíveis (SeloR)
